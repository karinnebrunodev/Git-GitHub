# Git e GitHub
## Aprendendo comandos Git
### Comandos básicos
#### git init

``` bash
git init
```
- Comando para iniciar um repositório Git

### git config
```bash
git config user.name 'nome do usuário'
git config user.email 'email do usuário'
```
- Comando para cofigurar credenciais de usuários

### git add
```bash
git add nome_do_arquivo
```
- Comando para adicionar arquivos a área de preparação para commit

### git commit
```bash
git commit -m 'mensagem de commit'
```
- Comando para adicionar um ponto na linha do tempo

### git remote
```bash
git remote add origin https://github.com/nome-de-usuario/nome-do-repositorio.git
```
- Comando para adicionar a origem remota do repositório

### git push
```bash
git push -u origin main nome-da-branch
```
- Comando para empurrar alterações locais para o repositório remoto
